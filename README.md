# ROBOSS

ROBOSS contains:

- customer frontend in [src](/C:/Users/PC/Downloads/roboare/robossoneman/src)
- HQ / branch admin frontend in [admin-dashboard](/C:/Users/PC/Downloads/roboare/robossoneman/admin-dashboard)
- backend API in [backend](/C:/Users/PC/Downloads/roboare/robossoneman/backend)

## Quick Start

1. Copy [backend/.env.example](/C:/Users/PC/Downloads/roboare/robossoneman/backend/.env.example) to `backend/.env`
2. Copy [.env.example](/C:/Users/PC/Downloads/roboare/robossoneman/.env.example) to `.env` for local frontend overrides
3. Run `cd backend && npm install && npm run verify:env`
4. Run `cd backend && npm run verify:production`
5. Run `cd backend && npm run verify:system`
6. Run `cd backend && npm run dev`
7. Run `npm install && npm run dev`
